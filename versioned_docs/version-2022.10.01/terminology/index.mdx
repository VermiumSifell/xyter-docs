# Terminology

## Guild {#Guild}

Represents a server on Discord.

## User {#User}

Represents a user on Discord.

## GuildMember {#GuildMember}

Represents a [User](#User) in a [Guild](#Guild)

## Module {#Module}

A module represents a specific feature in the bot, for example `credits` is an module as well as it's commands are modules such as `balance` and `top`.
