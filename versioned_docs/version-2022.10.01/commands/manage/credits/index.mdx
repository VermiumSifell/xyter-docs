# Credits

## Give

Allows administrators to increment a user credits.

## Giveaway

[![API - Controlpanel.gg](https://img.shields.io/badge/API-Controlpanel.gg-2ea44f?style=for-the-badge&logo=discord)](/docs/commands/config/cpgg)

Allows administrators to start a giveaway by creating a voucher with specified amount of uses and credits on [Controlpanel.gg](/docs/commands/config/cpgg) and then post the voucher in specified channel.

## Set

Allows administrators to set a user to a specific amount of credits.

## Take

Allows administrators to decrement a user credits

## Transfer

Allows administrators to transfer a specific amount of credits from one user to another.
