# Credits

**Enabled**: A boolean value that indicates whether the module should be enabled or not.
**Rate**: Specifies the amount of credits earned per message.
**Minimum Length**: Specifies the minimum length of a message to give credits to the author of the message.
**Work Rate**: Specifies the amount of maximum of credits a user can earn per workday.
**Work Timeout**: Specifies the length of a workday in seconds.
**Timeout**: Specifies the amount of time in seconds to wait before user can earn credits by writing messages again.
