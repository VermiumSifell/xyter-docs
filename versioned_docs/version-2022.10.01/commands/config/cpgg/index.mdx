# CPGG

[![API - Controlpanel.gg](https://img.shields.io/badge/API-Controlpanel.gg-2ea44f?style=for-the-badge&logo=discord)](/docs/commands/config/cpgg)

**Scheme**: Specifies if the bot should use `http://` or `https://` when communicating with the Controlpanel.gg API.  
**Domain**: Specifies the domain where the [Controlpanel.gg Panel](https://controlpanel.gg) lives.  
**Token**: Specifies the token provided by [Controlpanel.gg Panel](https://controlpanel.gg).
