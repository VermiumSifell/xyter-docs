# Introduction

Xyter is a bot built by [Zyner](https://zyner.org) and is designed with [privacy](/docs/features/privacy) in mind and is mainly developed using [TypeScript](https://www.typescriptlang.org/), [Discord.JS](https://discord.js.org), [Prisma](https://prisma.io).

It is **Free and Open Source Software** and licensed under the [GPL-3.0 license](https://github.com/ZynerOrg/xyter/blob/main/LICENSE).

## Core Features

- Supports multiple databases via [Prisma](https://prisma.io) such as SQLite and MySQL.
- Will forever remain as **Free and Open Source Software** with free support from the community.
- You can host the bot [On-Premise](/docs/flavors/on-premise) or use our [Cloud Instance](/docs/flavors/cloud).
- [Encryption](/docs/features/security/encryption) is available and is used for sensitive information such as API tokens.
- We are happy to help you customize the code and help you get the best experience with the bot.
- Most features are disabled by default ([why?](/docs/features/privacy/opt-in)), but is easily enabled via the [/manage](/docs/commands/manage) commands.
