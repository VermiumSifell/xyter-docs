# Introduction

Xyter is a privacy-focused bot with a huge **single-purpose**, replacing all bots with one. Xyter is Free & Open-Source and every feature __should__ be opt-in, as privacy is our main focus.

## Features

All our commands are implemented using **slash commands**.

You can find all our commands [here](/docs/commands).

### 3rd Party APIs

- [Controlpanel.gg](https://controlpanel.gg), used for buying [pterodactyl](https://pterodactyl.io/) servers.
